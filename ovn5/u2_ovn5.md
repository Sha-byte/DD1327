### Uppgift 2
#### Hur många kanter kan det som mest finnas i en graf med n stycken hörn?
Det kan finnas oändligt med kanter mellan hörnen i en graf, de kan finnas oändligt med kanter från ett hörn riktat på sig självt.
komplettering: Annars är det lika många som i b uppgiften dvs n(n-1)/2 plus en självloop för varje nod dvs n(n-1)/2 + n = n(n+1)/2.

#### Hur många kanter kan det som mest finnas i en enkel graf med n stycken hörn?
Det kan som mest finnas n över 2 kanter i en enkel graf med n hörn, dvs n*(n-1)/2.

#### Hur många kanter kan det som mest finnas i ett träd med n stycken hörn?
Det kan som mest finnas n-1 kanter i ett träd med n hörn
